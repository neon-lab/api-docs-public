Welcome

This is the documentation for Neon's API. You can use Neon's API to bring Neon's power of image processing and surfacing to your workflow and website.

The host for non-auth based requests is : https://services.neon-images.com

The host for auth based requests is : https://auth.neon-images.com

Topics

Auth Flow

We use JSON Web Tokens for authentication.
Learn more and get started interacting with our API in this section.

Authentication

Learn about the endpoints we make available for authentication.
Endpoints are available for refreshing tokens, authentications, and logouts.

CMS API

Use Neon's CMS API to create accounts, users, integrations, videos, and thumbnails.
There are additional GET, POST, and PUT endpoints described here.

Telemetry

Neon uses telemetry to understand how images are performing on your site and to make automated decisions based on video and image performance.
You'll find endpoints to view how images and videos are performing on your site here.
